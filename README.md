# Portfolio Site

## Getting Started

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Stack

- Static Site Generation - Next.js
- Style System - Chakra UI
- Build, Deployment, Hosting - Netlify
